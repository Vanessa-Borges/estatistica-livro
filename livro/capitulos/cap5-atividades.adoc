porém perdemos 8 reais para cada bola vermelha. Seja x:[X] o nosso lucro.
--
a) Determine a função de probabilidade de x:[X];

// x:[ p(-16) = 1/12, p(-7) = 1/6, p(2) = 13/36, p(11)=2/9, p(20) = 1/6].

// x:[E(X) = 4, Var(X) = 108,5].
